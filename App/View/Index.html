API - Encontro Gamer Pará de Minas

Recursos da API

Geral

Está API contém todos os métodos necessários para o correto funcinamento do CMS acoplado ao Site Encontro Gamer Pará de Minas. É possível realizar consultas no banco e operações de inserçoes e remoção. Todas as possibilidades de ajustes estão aqui listadas.

Requisições

As requisições devem ter os dados enviados no formato JSON, exceto quando for necessário o envio de arquivos, aceitando o FormType.
Requisições a endpoints públicos podem ser realizadas sem a informação de chaves de acesso.
Requisições a endpoints privados só aceitam requisições com um id de usuário (id_usuario) e seu token de acesso (st_token).

Login
[post] Login/

Recebe dados de usuário para realizar login no sistema.

[post] Login/[id_usuário]/logout

Destrói sessão de usuário no sistema

[post] Login/[id_usuário]/verificalogin

Verifica se um usuário tem sessões ativas no sistema

Usuario
[post] Usuario/

Altera informações de um usuário existente caso seja informado um id_usuario válido ou cria um novo usuário com as informações inseridas.

[get] Usuario/[id_usuário]

Retorna dados do usuário.

Jogo
[post] Jogo/

Altera informações de um jogo existente caso seja informado um id_jogo válido ou cria um novo jogo com as informações inseridas.

[delete] Jogo/[id_jogo]

Remove jogo informado da base de dados.

[get] Jogo/[id_jogo]

Retorna dados do jogo.

[get] Jogo/

Retorna dados de todos os jogos.

Imagem
[post] Imagem/

Altera informações de uma imagem existente caso seja informado um id_imagem válido ou cria uma nova iamgem com as informações inseridas. (Arquivo de imagem e o link gerado no ato de criação não podem ser alterado)

[get] Imagem/[id_imagem]

Retorna dados de imagem.

[get] Imagem/

Retorna dados de todas as imagens.

[delete] Imagem/[id_imagem]

Remove imagem informada.
